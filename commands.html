Commands

List of chat commands, sorted by category

Syntax explination
Here's an example command:
*example/example2 (argument 1) [argument 2]
*example/example2 is how the command is started, which includes one name (they are seperated by slashes) and a * before it.
(argument 1) is a required argument, this must be filled in.
[argument 2] this is not a required argument, and may be defaulted to something.
2 examples of this command's usage would be:
*example foo bar
*example2 foo



*sadcat: gets you a random sadcat from r/sadcats!

*blurry/blurrycat/blurrypictureofacat: gets a very blurry cat from r/blurrypicturesofcats

*8ball (question): Accurate telling of the future. Probably... (not)

*kick (@user) [reason]: kicks the mentioned user. If a reason is given, it will be shown in the bot's response and the audit log.

*ban (@user) [days of messages to delete, can be 0] [reason]: bans the mentioned user. Keep in mind that to show a reason, you must first give the days of messages to delete (this can be 0.)

*slowmode (off/time in seconds/time in 0h 0m 0s format): sets the slowmode for a channel. This can be adjusted with more freedom than the time presets Discord gives you.

*help [section or command]: Gives you a help menu, with sections and important commands. If an argument is given, it will list all commands in a given section, or go in detail about a given command.

*version: Gives the bot version, node.js version, and discord.js version
